## Sviluppo

Clona il progetto, installa le dipendenze e aggiorna i sottomuduli Git:

```bash
pnpm install
git submodule init
git submodule update
```

Per avviare un'istanza di sviluppo:

```bash
pnpm dev
```

## Compilazione

Per compilare un'istanza di produzione:

```bash
pnpm build
```

Per vederne un'anteprima:

```bash
pnpm preview
```
